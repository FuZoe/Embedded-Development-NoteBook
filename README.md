## 📖 使用说明

本项目包含嵌入式开发相关的学习笔记和参考资料，主要涵盖：
- **C++编程语言** - 面向对象编程、模板、STL等
- **CMake构建系统** - 跨平台构建、依赖管理、模块化开发
- **Qt框架开发** - GUI开发、事件处理、多线程、数据库集成
- **嵌入式系统开发** - 系统架构、驱动开发、性能优化

每个目录都包含相应的学习资料和示例代码，方便查阅和学习。
